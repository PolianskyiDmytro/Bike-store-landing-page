# 🚴 #Bike Store Landing Page

A responsive bike store landing page built according to a modern design concept.  
The project focuses on clean layout structure, responsive design, and scalable styling using BEM and SCSS.

---

## 🚀 #Features
- Fully responsive layout (desktop, tablet, mobile)
- Pixel-perfect implementation based on design
- Clean and modern UI for e-commerce presentation
- Structured sections (products, categories, about, contact)
- SCSS for scalable and maintainable styling
- BEM methodology for consistent CSS architecture
- Smooth and user-friendly layout

---

## 🛠️ #TechStack
- HTML5
- SCSS (Sass)
- JavaScript
- BEM methodology
- npm

---

## 🎯 #Purpose
This project was created to practice building modern landing pages, improve responsive design skills, and implement scalable CSS architecture in real-world UI scenarios.

### 📦 Install dependencies
npm install
# or
yarn install

### ▶️ Run the project locally
npm start
